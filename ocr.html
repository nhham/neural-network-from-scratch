Handwritten Optic Digit Recognition

Powered by a custom Python Neural Network built from scratch (NumPy)

How to use:

  1. Draw a digit (0-9) in the box below.
  2. Type the number in the "Target Digit" box to teach the AI.
  3. Click Train to update the "brain". The more you train, the more accurate!
  4. Click Test to see if the AI can recognize it!
